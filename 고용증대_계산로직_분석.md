## 4. 상시근로자 수 계산

### 제외 대상
- 근로계약기간 1년 미만
- 월 소정근로시간 60시간 미만 단시간근로자
- 임원
- 최대주주(대표자) 및 친족
- 근로소득세 원천징수·사회보험료 납부 미확인자

### 청년등 상시근로자
- 근로계약 체결일 현재 15세 이상 29세 이하
- 병역 이행 시 최대 6년 한도로 현재 연령에서 차감
- 장애인, 60세 이상 포함

### 인원수 산출 (종전 규정)
- `상시근로자수 = 총 근무개월수 합계 ÷ 12`
- 소수점 2자리까지 (버림)
- 엑셀에서는 소수 형태로 입력 (예: 7.58명)

## 5. 당기공제 계산 (1차년도)

### 증가 인원 산출
```
총 증가 = 당기 상시근로자 - 전기 상시근로자
청년 증가 = 당기 청년등 - 전기 청년등
청년외 증가 = 당기 청년외 - 전기 청년외
```

### 공제 대상 인원 (캡 적용)
```
공제대상 청년 = MIN(총증가, 청년증가)  (둘 다 양수일 때)
공제대상 청년외 = MIN(총증가 - 공제대상 청년, 청년외 증가)  (양수일 때)
```

> **핵심**: 청년 증가가 총 증가보다 크면 총 증가가 한도. 청년 우선 배정.

### 수도권/비수도권 분리 시 (우리 계산기)
```
수도권 합계 증가 = 청년수도권 증가 + 비청년수도권 증가
비수도권 합계 증가 = 청년비수도권 증가 + 비청년비수도권 증가

수도권 캡 = MAX(0, 수도권 합계 증가)
비수도권 캡 = MAX(0, 비수도권 합계 증가)

수도권 내 청년 우선: capYm = MIN(rawYm, metroCap)
수도권 내 비청년: capNm = MIN(rawNm, metroCap - capYm)
비수도권도 동일 로직
```

### 공제액
```
공제액 = INT(청년수도권 × 청년수도권단가)
       + INT(청년비수도권 × 청년비수도권단가)
       + INT(비청년수도권 × 비청년수도권단가)
       + INT(비청년비수도권 × 비청년비수도권단가)
```

> INT = 소수점 이하 버림 (TRUNC/FLOOR)

## 6. 추가공제 계산 (2차/3차년도)

### 유지 확인
확인연도의 상시근로자가 전기(기준연도) 대비 유지/증가했는지 판정.

### 청년 유지 시 (정상)
```
유지인원 = MIN(당기증가인원, MAX(0, 확인연도 - 전기))  (유형별)
공제액 = 유지인원 × 각 유형별 단가 (청년=청년단가, 비청년=비청년단가)
```

### 청년 감소 시
당기 대비 확인연도에 청년등 인원이 감소한 경우:
```
전체 유지인원 = MAX(0, 확인연도 전체 - 전기 전체)
유지비율 = MIN(당기총증가, 전체유지인원) / 당기총증가
각 유형 유지 = 당기증가인원 × 유지비율
공제액 = 모든 유형에 비청년 단가 적용
```

> **조세특례제도과-215 유권해석**: 엑셀에는 이 옵션을 on/off 토글로 제공.
> 우리 계산기에서는 청년감소→비청년단가 로직으로 반영 중.

### 3차년도 조건
- 2차년도에서 공제를 받았을 때만 3차년도 추가공제 가능
- 대기업은 3차년도 추가공제 없음 (years: 2)

## 7. 사후관리 (추징)

### 추징 발생 조건
최초 공제 과세연도 종료일부터 **2년** 되는 날이 속하는 과세연도 종료일까지의 기간 중 감소 시.

### 케이스 판정 (합산 기준)

| 케이스 | 조건 | 설명 |
|--------|------|------|
| **ok** | 전체 미감소, 청년 미감소 | 정상 — 추징 없음 |
| **나. (youth_only)** | 전체 유지/증가, 청년만 감소 | 청년→비청년 전환분 차액 추징 |
| **가.㉠ (case_a)** | 전체 감소, 청년감소 ≥ 전체감소 | 전환분 차액 + 이탈분 청년단가 추징 |
| **가.㉡ (case_b)** | 전체 감소, 청년감소 < 전체감소 | 청년분 + 비청년분 각각 추징 |

### 추징액 산출

#### 나. 청년만 감소
```
추징 = MIN(당기청년증가, 청년감소) × (청년단가 - 비청년단가)
```

#### 가.㉠ 전체감소, 청년감소 ≥ 전체감소
```
유효청년감소 = MIN(당기청년증가, 청년감소)
전환분 = MAX(0, 유효청년감소 - 전체감소)
이탈분 = 전체감소

추징 = 전환분 × (청년단가 - 비청년단가) + 이탈분 × 청년단가
```

#### 가.㉡ 전체감소, 청년감소 < 전체감소
```
추징 = MIN(청년감소, 전체감소) × 청년단가 + MIN(비청년감소, 전체감소) × 비청년단가
```

### 중소기업 수도권/비수도권 비율 배분
중소기업은 수도권/비수도권 단가가 다르므로 감소 인원의 지역 비율로 배분:
```
수도권청년비율 = 수도권청년감소 / (수도권청년감소 + 비수도권청년감소)
→ 각 비율에 따라 해당 지역 단가 적용
```

### 추징 승수 (mult)
- 2차년도 추징: mult = 1 (당기공제 1회분)
- 3차년도 추징: mult = 직전 2년 이내 공제받은 횟수 (최대 2)

### 이중추징 배제
```
3차년도 추징 = MIN(MAX(0, 산출추징 - 2차추징), MAX(0, 한도 - 2차추징))
```
> 2차년도에서 이미 추징한 금액은 3차년도에서 차감.

### 추징 한도
- 2차년도: 당기공제액
- 3차년도: 당기공제액 + 2차추가공제액

## 8. 검증 로직 ("처음부터 감소 상태로 계산")

최종 확인연도 기준으로 "처음부터 이 인원이었다면" 공제액을 역산하여 실제 순공제와 비교:

```
유효증가 = MIN(당기증가, MAX(0, 최종연도 - 전기))  (유형별)
청년감소 시 → 비청년단가 적용
연간공제 = 유효증가 × 단가
검증합계 = 연간공제 × 공제년수
실제순공제 = Σ(각 연도 공제액 - 추징액)

검증통과 = |검증합계 - 실제순공제| < 0.5
```

## 9. 엑셀 자동계산기 구조 (ver.4.3.6)

### 시트 구성
| 시트 | 용도 |
|------|------|
| 표지 | 입력 대시보드, 요약 결과, 3년간 공제/추징 요약 |
| 공제.추징 계산 | 핵심 계산 엔진 — 단가표, 1/2/3차 공제, 추징 수식 |
| 육아휴직등 관리 | 정규직전환/육아휴직 개인별 추적 (통합고용 전용) |

### 주요 입력 항목
- I2: 제도 선택 ("고용증대" / "통합고용")
- I5: 기업 규모 (중소/중견/대기업)
- L5: 지역 (수도권/수도권외) — **단일 지역 선택**
- N5: 유권해석 반영 여부
- T3: 추징 계산 방식 (기존 / 시행령)
- D8~D11: 연도 (0차~3차)
- G8~G11: 상시근로자수
- J8~J11: 청년등 인원

### 우리 계산기와의 차이
| 항목 | 엑셀 자동계산기 | 우리 웹 계산기 |
|------|----------------|--------------|
| 지역 입력 | 단일 선택 (수도권 OR 수도권외) | 4분류 동시 (수도권+비수도권 × 청년+비청년) |
| 시작연도 | 단일 (1개 타임라인) | 다중 (여러 시작연도 동시) |
| 유권해석 토글 | 있음 (반영/미반영) | 없음 (청년감소→비청년단가 자동) |
| 정규직전환/육아휴직 | 있음 (통합고용) | 없음 (고용증대에는 해당 없음) |
| 역사적 단가 | 2021-2022 자동 적용 | 2021-2022 자동 적용 (구현 완료) |
| 엑셀 출력 | 없음 | 있음 (ExcelJS) |
| 종합 매트릭스 | 없음 | 있음 (과세연도×시작연도 교차표) |
